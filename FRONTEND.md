# 🎨 Frontend - Baileys API Landing Page

## Apple Liquid Glass Design

Landing page inspirada no design **Liquid Glass** da Apple, apresentando todas as funcionalidades da API com efeitos de glassmorphism e animações suaves.

## 🚀 Como Executar

### Opção 1: Frontend apenas
```bash
npm run frontend
```
Acesse: http://localhost:5173

### Opção 2: API + Frontend simultaneamente
```bash
npm run dev:full
```
- API: http://localhost:3000
- Frontend: http://localhost:5173

### Opção 3: Build para produção
```bash
npm run frontend:build
npm run frontend:preview
```

## 🎨 Funcionalidades da Landing Page

### Design Apple Liquid Glass
- ✅ Efeitos de glassmorphism com backdrop blur
- ✅ Animações fluidas com Framer Motion
- ✅ Elementos flutuantes no background
- ✅ Transparências e gradientes dinâmicos
- ✅ Botões com efeito "liquid" interativo

### Seções da Página
- **Hero Section**: Título principal com estatísticas
- **Features Grid**: Cards interativos mostrando funcionalidades
- **API Endpoints**: Documentação expandível por funcionalidade
- **Quick Start**: Guia de início rápido
- **Footer**: Links para documentação

### Funcionalidades Demonstradas
1. **Multi-Sessões**: Gerenciamento de múltiplas conexões
2. **Mensagens Inteligentes**: Comportamento humano simulado
3. **Grupos**: Controle completo de grupos WhatsApp
4. **Webhooks**: Sistema de até 3 webhooks por sessão

## 🛠️ Tecnologias

- **Vite** - Build tool ultra-rápido
- **React** - Interface reativa
- **Tailwind CSS** - Estilização utilitária
- **Framer Motion** - Animações suaves
- **Headless UI** - Componentes acessíveis
- **Heroicons** - Ícones consistentes
